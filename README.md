# Multi-Agent-Chatbot

## Requirements:
    Docker
    Python3

## How-tos:

### Run the app & ollama servers
    Inside the Project 2 folder:
    ```
    docker compose up --build -d
    ```

### To stop the servers
    ```
    docker compose down -v
    ```

### Documentation
    After you run the servers: [a link](http://localhost:80/docs)
